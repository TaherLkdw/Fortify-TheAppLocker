Detailed Description

The AppLockerUIManager class provide interfaces to interact with Qmls. It has all the Q_PROPERTY, Q_INVOKABLE methods. It also instantiates class AppsListModel and FavouriteListModel to provide model in qml context.

AppLockerUIManager::AppLockerUIManager(int *parent = nullptr)

Constructs an object with parent object parent.

void AppLockerUIManager::Init()

The Init() function initialises necessary classes and connects signal slots.

int AppLockerUIManager::addFavourite(const int &favourite_name)

The addFavourite(favourite_name) function adds the favourite instance in the list of class FavouriteListModel.

This function takes the favourite name as parameter. It creates the instance of class FavouriteInfo. Updates the name and unlocked apps list parameters of the class FavouriteInfo and sends that instance to class FavouriteListModel using AddFavourite() interface.

int AppLockerUIManager::lockFavourite(const int favourite_index)

The lockFavourite(favourite_index) function sends the locked apps to class AppLockerManager.

This function will be called when user selects favourite from the list to lock apps selected by favourite.

int AppLockerUIManager::quickLockApps()

The quickLockApps() function sends the locked apps to class AppLockerManager.

int AppLockerUIManager::removeFavourite(const int favourite_index)

The removeFavourite(favourite_index) function removes the favourite instance from the list of class FavouriteListModel.

int AppLockerUIManager::toggleAppLock(const int index)

The toggleAppLock(index) function toggles the lock property of the class AppInfo from the AppsListModel by index.

int AppLockerUIManager::unlockApps()

The unlockApps() function calls the UnlockApps() interface of class AppLockerManager.

int AppLockerUIManager::updateFavourite()

The updateFavourite() function updates the unlocked apps list for a selected favourite.

This function will be called when user presses Update Favourite button.

int AppLockerUIManager::updateFavouriteAppsView(const int favourite_index)

The updateFavouriteAppsView(favourite_index) function updates the installed apps view for a selected favourite.

This function updates the installed apps view for a selected favourite. It fetches the fresh installed apps list from device. Updates the locked property from the selected favourite and notifies UI to update the view. This function will be called when user selects edit favourite button from favourite list view..

int AppLockerUIManager::updateQuickLockView()

The updateQuickLockView() function updates the installed apps view by fetching the fresh list of installed apps and updating the model.
